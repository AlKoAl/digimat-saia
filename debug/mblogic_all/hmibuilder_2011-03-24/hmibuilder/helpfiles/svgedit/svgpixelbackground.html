Adding Background Pixel Graphics

It is occassionally useful to add pixel graphic images (e.g. PNG files) to drawings (e.g. for background images). This is useful for example when you don't have an SVG version available to you.

To do this, simply drag the image file onto Inkscape and position it where you want it. If this is intended to provide a background to all the layers, then you may want to put it in the background layer, on top of the background rectangle.

This however only places a reference to the image file into the drawing. It does not copy the file itself. You will need to place a copy of the image file together with the finished web page in the application web page directory so that it can be served with the other resources when the web browser requests it.
