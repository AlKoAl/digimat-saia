Pipes, Ducts, and Conveyors

The provided pipes are essentially the same as pilot lights. See the pilot light documentation for more details on that topic.

Pipes, ducts, conveyors, and other similar devices can also be created by simply drawing rectangles or other closed. They can then be manipulated as pilot lights via their "id". Multiple rectangles can be grouped together by using the "group" feature.

When creating the rectangles the "stroke" property should be set to "none". This turns off the outline around the rectangle, which in turn prevents the "joints" between different rectangles from being visible.

See the section on user created graphics for addition points to consider.
