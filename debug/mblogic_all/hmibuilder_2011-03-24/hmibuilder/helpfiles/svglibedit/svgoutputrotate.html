Rotating Output Objects

Rotating output objects are graphics whose "rotate" property is controlled. These include dial gauges and rotating pumps.

Dial Gauges

Dial gauges consist of a dial background which is simply a passive background object, plus a pointer which is rotated to indicate a relative value.

  1. Drag a dial background onto the screen.
  2. Drag a pointer onto the screen and position it on top of a dial background.
  3. Select the pointer object.
  4. Right click and select "object properties".
  5. Set the "id" to the desired name. Be sure to record this name as you will need it later.

Rotating Pumps

Rotating pumps consist of a pump frame background which is simply a passive background object, plus a rotor which is rotated continuously (incrementally) to indicate a state.

  1. Drag a pump frame background onto the screen.
  2. Drag a rotor onto the screen and position it on top of a pump background.
  3. Select the rotor object.
  4. Right click and select "object properties".
  5. Set the "id" to the desired name. Be sure to record this name as you will need it later.
