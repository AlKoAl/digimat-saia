Strip Charts

Strip charts are polylines whose points define a series of lines. Each cycle, the polyline is updated with a new set of points.

  1. Draw a suitable background, including axes.
  2. Drag a strip chart onto the drawing.
  3. Set the "id" property and record it for later use.
  4. Do not stretch the polyline to fit the graph. The scaling factors are set elsewhere. Stretching the line will also expand the width and add a position offset.
