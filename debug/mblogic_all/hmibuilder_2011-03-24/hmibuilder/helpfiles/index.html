Help - Creating and Configuring the HMI

Drag and Drop Graphical HMI Development:

The graphical portion of the HMI can be created using the Inkscape vector graphics editor.

The provided graphical widgets (push buttons, pilot lights, etc.) can be used as is and animated, or you can draw your own custom graphics and animate those, or mix both methods together.

  • Editing SVG with Inkscape - Using Inkscape as a graphical editor.
  • SVG Library - The SVG graphical widget library.
  • Using the SVG Library with Inkscape - Using Inkscape to create HMI web page graphics.
  • Custom Widgets - Creating custom widgets.

Assembling Pages Using HMIBuilder

You can assemble the web page and create the scripting automatically by using the HMIBuilder program.

  • HMI web templates - A description of the provided templates.
  • Assembling web pages - Assembling HMI web pages from templates.
