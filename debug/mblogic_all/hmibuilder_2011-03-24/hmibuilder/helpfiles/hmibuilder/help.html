Help - Using the Help System

Help System

The on line help system is web based. Selecting the "help button" will cause the following things to happen:

  1. The help web server will be started if it is not already running.
  2. Your system's default web browser will be started if it is not already running.
  3. Your default web browser will be asked to load the main help web page.

All the on-line help will then be available via your web browser.

Help Web Server

The "help" system web server uses port 8880. If it is not able to bind to this port, the help system will not be available.
