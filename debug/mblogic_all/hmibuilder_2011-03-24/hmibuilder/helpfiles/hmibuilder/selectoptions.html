Select Options

This step selects the overall drawing options. This consists of the following:

  • Layers - This will scan the SVG drawing for any layers which may exist. It will automatically assign ids to those layers which match the layer names which you assigned in the drawing (Inkscape layers names are separate from the actual ids). You will need to select which layers you wish to use in the SVG menu system (if you are using multiple "screens"). Be careful which layers you select. Do not select the layer you have your menu buttons on! This information will be used to generate the "screen list" used by the SVG menu system.
  • Alarm and event zones - This will allow you to select which alarms and events you wish to monitor in this HMI web page. Any alarm or events zones which are not selected will not be monitored. This information is used to generate the the alarm and event zone polling lists. The zone names are read from the hmi configuration file which you selected in a previous step.

This information will be stored inside the SVG drawing when you save the SVG file after editing it. This will allow you to save your work and resume editing at a later time if you wish.

Steps

  1. Check the layers you wish to use as part of the SVG screen menues. These are the drawing layers you created. Select the layers that you wish to have controlled as part of the screen menu system. Do not select layers that you do not wish to have controlled by the menu system.
  2. Select the alarm zones that you wish to monitor.
  3. Select the event zones that you wish to monitor.
  4. You may at this step save your work by selecting the "Save" button.
  5. Click on the "Next" button before proceeding to the next step.
