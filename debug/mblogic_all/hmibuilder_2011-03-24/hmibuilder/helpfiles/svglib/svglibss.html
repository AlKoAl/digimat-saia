Selector Switches

Selector switch labels are essentially push buttons which write a value to the server when clicked. The provide the main input functionality of a selector switch.

Selector switches also have input functions, but these differ between two and three position switches. The actuator position of the switch acts as an output and must be controlled via the SVG "id".

Type Name Description HMIBuilder Name
Selector switch label. ss2_label_<colour> Selector switch label / push button for a two position selector switch. Two of these are normally combined with a two position selector switch to form a selector switch assembly. Selector Switch Label
Two position selector switch. ss2_short_<colour> Two position selector switch actuator. When the switch is clicked, the input state will toggle. The rotation angle of the actuator is controlled via the SVG "id" and acts as an output. 2 Pos SS
Selector switch label. ss3_label_<colour> Selector switch label / push button for a three position selector switch. Three of these are normally combined with a three position selector switch to form a selector switch assembly. Selector Switch Label
Three position selector switch. ss3_short_<colour> Three position selector switch actuator. When the switch is clicked, the input state will be set to a user specified value. The rotation angle of the actuator is controlled via the SVG "id" and acts as an output. 3 Pos SS

HMIBuilder Parameters for Two Position Selector Switch

When the switch itself is clicked, it will write a new value to the HMI server. If the value of the reference tag is 0, it will write a 1, if the reference value is not 0, it will write a 0. If the switch tag is 0, the switch will be rotated to the left. if the switch tag is not 0, the switch will be rotated to the right.

Parameter Type Values Description
Write Tag Option menu Address tags The HMI server tag to write to.
Reference Tag Option menu Address tags The reference tag.
Switch Tag Option menu Address tags The tag to use to indicate the switch state.

HMIBuilder Parameters for Three Position Selector Switch

This is similar to a two position selector switch, with the following exceptions. Clicking on the switch will always write a 0, rather than toggling the value. The switch position is determined as follows. If the value of the switch tag is negative, the switch will be rotated to the left. If the value of the switch tag is zero, the switch will be rotated to the centre position. If the value of the switch tag is positive, the switch will be rotated to the right.

Parameter Type Values Description
Write Tag Option menu Address tags The HMI server tag to write to.
Switch Tag Option menu Address tags The tag to use to indicate the switch state.
