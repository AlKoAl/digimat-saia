Manual HMI Web Page Construction

Overview

This section describes how to create HMI client web pages manually using traditional web page authoring techniques. This will be of interest to those who are familiar with manual web page creation and who wish to create custom HMI web pages.

Topics

  • Page Layout - An example exaplaining all the parts of a simple HMI web page and where they go.
  • Positioning Components on the Screen - How to position, rotate, and scale SVG elements on the screen.
  • Communications, the Display List and Data Sources - How the client and server communicate and update the screens.
  • Menu Control - Controlling the display of screens using HTML menus.
  • Events, Alarms, and System Status - Displaying alarms, events, and communications errors.
  • Mixing HTML and XUL with SVG - HTML and XUL.
  • Push Buttons - Creating push buttons.
  • Pilot Lights - Creating pilot lights.
  • Numeric and Text Displays - Creating numeric and text data entry displays.
  • Tanks and Gauges - Creating various tanks and gauges.
  • Piping and Pumps - Creating different pipes, pumps, and other continuous animations.
  • Numeric Inputs - Creating numeric data entry devices.
  • Strip Charts - Basic strip charts.
  • Background Images - Inserting static background images into a screen.
  • Visual Effects - How to create simple visual effects.
